這是一個融合了戰術拆彈（如《絕對武力》）與MOBA英雄機制（如《英雄聯盟》）的 2D 俯視角（Top-down）或 2D 橫向捲軸（Side-scroller）多人線上攻防遊戲。
我們將這款遊戲命名為：《霓虹重組》（Neon Overdrive: Breach）
## 遊戲核心設定 (Core Concept)

視角與畫面： 2D 高精細像素風（Pixel Art）或賽博龐克霓虹向量風。採用俯視角（Top-down），具有嚴格的戰爭迷霧（Fog of War）與視線（Line of Sight）系統——障礙物後方與轉角是完全看不到的，極度依賴聽覺與探測道具。
對決人數： 5 v 5 團隊對抗。
勝利條件（回合制）：
攻擊方（駭客陣營 - Breachers）： 在指定區域（A 點或 B 點）部署「數據解密器（Data Decryptor）」，並保護其至解密完成；或全殲敵方。
防守方（維安陣營 - Keepers）： 阻止解密器被部署、在時限內拆除已部署的解密器；或全殲敵方。 ## 核心玩法與機制 (Gameplay Mechanics)
獨特的「2D 戰術空間」
因為是 2D 俯視角，遊戲非常強調「卡位」、「彈道掩護」與「視野盲區」。

可破壞牆面： 部分牆壁可以用重型武器或炸藥炸開，創造新的進攻路線（類似 2D 版的《虹彩六號》）。
聲音視覺化： 跑步、開槍、裝彈都會在戰爭迷霧中向敵方發出短暫的「波紋提示」。靜步走（Shift）則不會發出波紋。
英雄與經濟系統
玩家在每局開始前可以選擇不同定位的英雄（Agent），每位英雄擁有一個被動技能、兩個可購買的戰術技能，以及一個需要充能的大招。

先鋒/突破位（Duelist）： 擁有位移、閃光彈或瞬時護盾，負責撕裂防線。
守衛/陣地位（Sentinel）： 能放置 2D 雷射絆線、自動機槍塔或封鎖線，減緩敵人進攻。
情報/偵察位（Initiator）： 發射聲納箭、無人機，在迷霧中開拓視野。
控場/煙幕位（Controller）： 投擲數位煙幕（阻擋 2D 視線）、電漿牆，分割戰場。
每回合依據勝敗、擊殺、拆除表現獲得「比特幣（Credits）」，用於購買下一回合的武器（衝鋒槍、散彈槍、突擊步槍、狙擊電磁砲）與技能次數。
## 地圖設計與攻防節點
典型地圖包含 三個主要區域：A 區、B 區 以及中央爭奪區（Mid）。

通道與掩體： 地圖充滿了箱子、通風管（只有特定矮小體型或技能能通過）和電子閘門（可被駭客遠端開啟或被維安鎖死）。
垂直高低差（若採 Top-down 則為平台機制）： 某些高台需要走樓梯或使用鉤爪上去，高處對低處擁有視野優勢，但容易暴露在沒有掩體的地方。
## 精彩攻防情境範例 (Gameplay Scenario)
場景：A 據點，回合剩餘 45 秒，3 v 2（防守方人數優勢）

進攻方（Breachers）僅剩兩人，攜帶解密器的「暗影（Shadow）」與負責掩護的「重砲（Titan）」。
「暗影」施放技能【數位煙幕】，一團黑色粒子瞬間籠罩了 A 點入口，阻擋了防守方十字交火的視線。
防守方（Keepers）的偵察兵立刻向煙幕內發射【感應飛鏢】，2D 畫面上立刻以紅色輪廓勾勒出「暗影」正蹲在地上安裝解密器！
防守方的重裝兵看準輪廓，拿起機槍穿牆掃射（Bullet Penetration）。
進攻方的「重砲」果斷上前，開啟【電漿護盾】擋在「暗影」身前，用肉身硬扛子彈。
「解密器已部署！」 語音響起。
此時戰局逆轉，防守方必須在 45 秒內衝進去拆除，而進攻方則利用殘存的掩體和陷阱，展開驚心動魄的 2D 彈幕防守……
## 遊戲的獨特賣點 (USP)

低配備、高競技性： 2D 畫面讓低階電腦或手機也能流暢 120 幀運行，但極致的視線系統讓競技深度不輸 3D 大作。
觀賽視角極佳： 對於觀眾與主播而言，2D 全知視角（上帝視角）能一眼看清兩隊的戰術走位、包抄與心理戰，極具電競潛力。按esc退出遊戲
